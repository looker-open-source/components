Flexbox is perfect for aligning items inside components. It typically isn't used for large scale layouts but for smaller parts of a page or component. All of the flex box utility classes can be used with our breakpoint suffixes.

Flex Container

The flex container is the parent of the items that will be laid out using flex box. The properties in this section are all applied on this container.

Display Properties

The display property defines the flex container and enables a flex context for all of its direct children.

@extend Resolves to... Notes
%flex-block display: flex Used on block elements
%flex-inline display: flex-inline Used on inline elements

Flex Direction

This establishes the main-axis, defining the direction items are palced in the container. Flex lays out in a single direction, either horizontal or vertical.

@extend Resolves to... Notes
%flex-row flex-direction: row (Default) Left to right.
%flex-row-reverse flex-direction: row-reverse Right to left
%flex-column flex-direction: column Top to bottom
%flex-column-reverse flex-direction: column-reverse Bottom to top

Flex Wrap

Flex items will all fit into one line by default. To change that behavior and allow them to wrap, you'll need to add some properties. Flex direction plays a role here by defining the direction new line are stacked.

@extend Resolves to... Notes
%flex-nowrap flex-wrap: nowrap (Default) Single-line with no wrapping
%flex-wrap flex-wrap: wrap Multi-line from left to right
%flex-wrap-reverse flex-wrap: wrap-reverse Multi-line from right to left

Justify Content

This defines the alignment along the main axis and helps distribute space around the flex items.

@extend Resolves to... Notes
%flex-justify-start justify-content: flex-start (Default) Items are packed toward the start line
%flex-justify-end justify-content: flex-end Items are packed toward the end line
%flex-justify-center justify-content: center Items are centered along the line
%flex-justify-between justify-content: space-between Items are evenly distributed in the line. First item on the start line, last item on the end line.
%flex-justify-around justify-content: space-around Items are evently distributed in the line with equal space around them. Note that visually the spaces aren't equal because each item has equal space on both sides.

Align Items

This defines how flex items are laid out along the cross axis on the current line.

@extend Resolves to... Notes
%flex-item-stretch align-items: stretch (Default) Stretch to fill the container but still respects min/max-width
%flex-item-start align-items: flex-start Cross-start margin edge of the items is placed on the cross-start line
%flex-item-end align-items: flex-end Cross-end margin edge of the items is placed on the cross-end line
%flex-item-center align-items: center Items are centered in the cross-axis
%flex-item-baseline align-items: baseline Items are aligned along their baselines

Align Content

This aligns a flex container's lines within when there is extra space in the cross-axis, similar to how justify-content aligns items within the main axis. Note: this property has no effect when there is only one line of flex items.

@extend Resolves to... Notes
%flex-content-stretch align-content: stretch (Default) Lines stretch to take up the remaining space
%flex-content-start align-content: flex-start Lines packed to the start of the container
%flex-content-end align-content: flex-end Lines packed to the end of the container
%flex-content-center align-content: center Lines packed to the center of the container
%flex-content-around align-content: space-around Lines evenly distributed with the first line at the start and the last at the end of the container.
%flex-content-between align-content: space-between Lines evenly distributed with equal space around each line.

Flex Items

These are the direct children of the parent container. Flex items are laid out according to the properties applied to the flex container. The following properties are applied on the individual flex items.

Order

By default, items are laid out in source order. Use the order property to control the order inside the flex container.

Lens contains built in ordering classes of .flex-order-$n, where $n is an integer from 1 to 6. Any item without an order will default to 1 and matching orders will be grouped together in the appropriate order. You can rearrange the order across breakpoints using the breakpoint suffixes.

@extend Resolves to... Notes
%flex-order-$n order: $n $n is a an integer from 1 to 6

Flex Grow

This property lets an item grow if necessary. It accepts a unitless value that serves as a proportion. The default is 0.

@extend Resolves to... Notes
%flex-grow-$n flex-grow: $n $n is a an integer from 0 to 6

Flex Shrink

This property lets an item shrink if necessary. It also accepts a unitless value that serves as a proportion. The default is 1.

@extend Resolves to... Notes
%flex-shrink-$n flex-shrink: $n $n is a an integer from 1 to 6

Flex Basis

This property defines the default size of an element before the remaining space is distributed. It can be a length or keyword, like "auto". The default is auto.

To use percentages, use the same percentage values used to layout grids (5-100 in increments of 5, plus 33 and 66). To use a sizing unit preface the unit with an s.

@extend Resolves to... Notes Example
%flex-basis-auto flex-basis: auto Default
%flex-basis-$n flex-basis: percentage($n/100) $n is a an integer from 5-100 in increments of 5, plus 33 and 66 %flex-basis-33 resolves to flex-basis: 0.33
%flex-basis-s$size flex-basis: s$size $size is a sizing unit %flex-basis-s4 resolves to flex-basis: 4px

Align Self

This allows individual flex items to have their own unique alignment within the container overriding the container align-items property.

@extend Resolves to... Notes
%flex-self-auto align-self: auto (Default) Auto align, which inherits the align-items property
%flex-self-start align-self: flex-start Cross-start margin edge of the items is placed on the cross-start line
%flex-self-end align-self: flex-end Cross-end margin edge of the items is placed on the cross-end line
%flex-self-center align-self: center Items are centered in the cross-axis
%flex-self-stretch align-self: stretch Stretch to fill the container but still respects min/max-width
%flex-self-baseline align-self: baseline Items are aligned such as their baselines align
