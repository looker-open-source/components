`<Tabs2>` expects to receive one or more `Tab2` components. `Tabs2` manages the visibility of the single `content` based on its internal `Tab2` content.

`Tab2` expects to have `label` that will be the value displayed on the `Tab` itself. The value passed as children will be displayed as its content area.

```jsx
<Tabs2>
  <Tab2 id="cats" label="Cats">
    Here's awesome story about cats
  </Tab2>
  <Tab2 id="dogs" label="Dogs">
    Cats are way better than dogs. Go to other tab
  </Tab2>
  <Tab2 label="Fish">Are kinda smelly</Tab2>
</Tabs2>
```

## Horizonal Scroll

`Tabs` supports scrolling when the tabs' combined width cannot be accomdated within the viewport width. - Try scrolling left-right.

```jsx
<Tabs2>
  <Tab2 label="1">
    Really long tab title that should help demonstrate side-scroll
  </Tab2>
  <Tab2 label="2">Yet-another-tab-title to get things width enough</Tab2>
  <Tab2 label="3">
    Really, you must be obsessed with these long tab titles?
  </Tab2>
  <Tab2 label="4">
    Okay, okay, we get it you want to see scrolling in action...
  </Tab2>
</Tabs2>
```

## Disabled Tab(s)

To create a disabled Tab add the `disabled` prop to each individual `Tab2`.

```jsx
<Tabs2>
  <Tab2 id="cats" label="Cats">
    Here's awesome story about cats
  </Tab2>
  <Tab2 id="dogs" label="Dogs">
    Cats are way better than dogs. Go to other tab
  </Tab2>
  <Tab2 label="Fish">Are kinda smelly</Tab2>
  <Tab2 disabled label="Human">
    not available
  </Tab2>
</Tabs2>
```

## Distribute Attribute

Using the distribute prop on `Tabs2` will equally distribute each `Tab` in the container.

```jsx
<Tabs2 distribute>
  <Tab2 label="Rebecca Purple">
    Rebecca Purple{' '}
    <div
      style={{
        backgroundColor: 'RebeccaPurple',
        height: '350px',
      }}
    />
  </Tab2>
  <Tab2 label="Coral">
    {' '}
    <div
      style={{
        backgroundColor: 'coral',
        height: '350px',
      }}
    />
  </Tab2>
  <Tab2 label="Forest Green">
    {' '}
    <div
      style={{
        backgroundColor: 'ForestGreen',
        height: '350px',
      }}
    />
  </Tab2>
  <Tab2 label="Cornflower Blue">
    {' '}
    <div
      style={{
        backgroundColor: 'CornflowerBlue',
        height: '350px',
      }}
    />
  </Tab2>
</Tabs2>
```

## Controlled Tab

```jsx
;() => {
  const [currentTabIndex, setTab] = useState(0)

  return (
    <>
      <Space my="small">
        <Button onClick={() => setTab('cats')}>check out cats</Button>
        <Button onClick={() => setTab('dogs')}>check out dogs</Button>
      </Space>
      <Divider />
      <Tabs2 tabId={currentTabIndex} onTabChange={setTab}>
        <Tab2 id="cats" label="Cats">
          Here's awesome story about cats
        </Tab2>
        <Tab2 id="dogs" label="Dogs">
          Cats are way better than dogs. Go to other tab
        </Tab2>
      </Tabs2>
    </>
  )
}
```
