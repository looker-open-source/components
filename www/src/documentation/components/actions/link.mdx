title: Link
---

The `<Link />` component renders an `<a>` tag that accepts and `href` property. You can also supply an optional `id` property if you want to give your anchor an id.

```jsx
<UnorderedList>
  <li>
    <Link href="https://looker.com">ğŸ‘‹ I am a link!</Link>
  </li>
  <li>
    <Link href="https://looker.com" id="thumbs-up">
      ğŸ‘ I am a link with an id
    </Link>
  </li>
</UnorderedList>
```

## Underline

By default, the underline appears on hover, focus, active, and visited states. To disable the underline always, set the `underline` prop to `false`. To enable it always, set it to `true`.

```jsx
<UnorderedList>
  <li>
    <Link href="https://looker.com">Underline undefined</Link>
  </li>
  <li>
    <Link href="https://looker.com" underline={false}>
      Underline false
    </Link>
  </li>
  <li>
    <Link href="https://looker.com" underline>
      Underline true
    </Link>
  </li>
</UnorderedList>
```
