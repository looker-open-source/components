`TextArea` component offeres a multi-line text editor. If you are bulding a form `FieldTextArea` may be a better fit.

```jsx
<TextArea resize="vertical" />
```

Disabled

```jsx
<TextArea disabled />
```

Error messeges

```jsx
<TextArea validationType="error" />
```
