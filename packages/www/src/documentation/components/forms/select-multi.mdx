The `SelectMulti` component is an extension to the `Select` component that supports multiple selected values and has the following specific features:

- Renders in the style of `InputChips`
- Option list stays open by default so the user can select multiple options (see `closeOnSelect` prop)
- An optional on-the-fly create option allows the user to enter custom values (see `showCreate` prop)
- Values can be cleared individually or all at once (does not use the `Select` component's `isClearable` prop)
- Aside from `isClearable`, all features of `Select` are supported in `SelectMulti`

## Basic

```jsx
<SelectMulti
  options={[
    { value: 'Cheddar' },
    { value: 'Gouda' },
    { value: 'Swiss' },
    { value: 'Feta' },
    { value: 'Mascarpone' },
    { value: 'Brie' },
    { value: 'Mozzarella' },
    { value: 'Cotija' },
    { value: 'Pepperjack' },
  ]}
  placeholder="Cheeses"
/>
```
