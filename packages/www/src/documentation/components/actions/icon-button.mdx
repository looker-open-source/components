title: IconButton

`IconButton`'s are used when you need an icon to trigger an action, such as showing a dialog or deleting an item. Icon buttons require a `label` that describes it's action. The `label` should be descriptive and succinct.

`IconButton`'s come in five sizes: `xxsmall`, `xsmall` (default), `small`, `medium`, and `large`. It also supports two shapes: `round` and `square`.

If you need an icon for purely decorative purposes use an `<Icon />` instead.

```jsx
;() => {
  const ButtonCollection = styled.div`
    display: grid;
    grid-auto-flow: column;
    grid-gap: ${({ theme }) => theme.space.medium};
    justify-items: center;
    align-items: center;
  `

  return (
    <ButtonCollection>
      <IconButton label="Add Something Neat" icon="Plus" size="xxsmall" />
      <IconButton label="Settings" icon="Gear" />
      <IconButton label="Add to Favorites" icon="Favorite" size="small" />
      <IconButton
        label="Delete not-so-great Thing"
        icon="Trash"
        size="medium"
        color="danger"
      />
      <IconButton
        label="Forward"
        icon="CaretRight"
        size="large"
        shape="round"
        outline
      />
    </ButtonCollection>
  )
}
```

```jsx
;() => {
  const HoverVariantButton = styled(IconButton)`
    transition: transform 0.5s;
    &:hover {
      transform: rotate(45deg);
    }
  `
  return (
    <HoverVariantButton
      label="Close"
      icon="Close"
      size="large"
      outline
      mr="small"
    />
  )
}
```

You can also use your own `Tooltip` around `IconButton` (this will effectively remove the "built-in" `Tooltip`)

```jsx
<Tooltip content="I'm a little teapot">
  {(eventHandlers, ref) => (
    <IconButton
      {...eventHandlers}
      ref={ref}
      label="Add Something Neat"
      icon="Plus"
      size="xxsmall"
    />
  )}
</Tooltip>
```
