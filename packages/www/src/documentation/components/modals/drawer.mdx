Drawers are modal overlays that provide contextual information. They are frequantly used for creation or management action by the user. The Drawer blocks interactions with the other content until dismissed either through clicking or tapping on the Backdrop or by selecting a close or completion action within the Drawer.

The Drawer component allows you to manage state externally.

```js
;() => {
  const { value, setOn, setOff } = useToggle()

  return (
    <div>
      <Drawer isOpen={value} onClose={() => setOff()}>
        <ModalContent>
          <Paragraph>Some content inside the Drawer</Paragraph>
          <Button>A button too!</Button>
        </ModalContent>
      </Drawer>
      <Button onClick={() => setOn()}>Open Drawer</Button>
    </div>
  )
}
```

## DrawerManager

<Props of="DrawerManager" />

DrawerManager gives an easy way to compose a Drawer without the need to manage state.

```jsx
<DrawerManager
  content={
    <ModalContent>
      <p>Some content</p>
      <Button>A button too!</Button>
    </ModalContent>
  }
>
  {onClick => <Button onClick={onClick}>Open Drawer</Button>}
</DrawerManager>
```

### Composed Drawer

Drawers can be constructed with Modal helpers to create a complex layout a fixed-position header and footer as well as a content area that indicates overflow and allows the user to scroll through its contents.

```jsx
<DrawerManager
  content={
    <React.Fragment>
      <ModalHeader>
        <Heading>Some Things to Look at</Heading>
      </ModalHeader>
      <ModalContent>
        <Box height="40vh" bg="lavender" mb="large">
          Thing One
        </Box>
        <Box height="40vh" bg="lavender" mb="large">
          Thing Two
        </Box>
        <Box height="40vh" bg="lavender" mb="large">
          Thing Three
        </Box>
      </ModalContent>
      <ModalContext.Consumer>
        {({ closeModal }) => (
          <ModalFooter>
            <Button
              onClick={() => {
                alert("Oh yeah! You're done")
                closeModal()
              }}
            >
              All Done
            </Button>
            <ButtonTransparent onClick={closeModal}>
              Cancel
            </Button>
          </ModalFooter>
        )}
      </ModalContext.Consumer>
    </React.Fragment>
  }
>
  {onClick => (
    <Button onClick={onClick} variant="outline">
      Drawer with sample content
    </Button>
  )}
</DrawerManager>
```
