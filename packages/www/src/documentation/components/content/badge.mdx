title: Badge
propsOf: Badge
---

`Badge` can be styled for size, color and it has flags for round border and transparency.

## Badge Sizes

```jsx
<Badge m="small">small</Badge>
<Badge m="small" size="medium" >
  medium
</Badge>
<Badge m="small" size="large" >
  large
</Badge>
```

## Badge Intent

```jsx
<Badge intent="plain" m="small" size="medium" >
  plain
</Badge>
<Badge intent="positive" m="small" size="medium" >
  positive
</Badge>
<Badge intent="info" m="small" size="medium" >
  info
</Badge>
<Badge intent="neutral" m="small" size="medium" >
  neutral
</Badge>
<Badge intent="warning" m="small" size="medium" >
  warning
</Badge>
<Badge intent="critical" m="small" size="medium" >
  critical
</Badge>
```

## Badge Flags

```jsx
<Badge m="small" size="medium" transparent >
  transparent
</Badge>
<Badge m="small" round size="medium" >
round
</Badge>
```
