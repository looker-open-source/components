title: Badge
propsOf: Badge

`Badge` has its default values as size small and background color as purple. The styles can be updated for size, color, and it has a flag for `rounded` border.

## Badge Sizes

```jsx
<Flex alignItems="center" justifyContent="space-around">
  <Badge size="small">small</Badge>
  <Badge size="medium">medium</Badge>
  <Badge size="large">large</Badge>
</Flex>
```

## Badge Intent

```jsx
<Flex alignItems="center" justifyContent="space-around">
  <Badge intent="plain">plain</Badge>
  <Badge intent="positive">positive</Badge>
  <Badge intent="info">info</Badge>
  <Badge intent="neutral">neutral</Badge>
  <Badge intent="warning">warning</Badge>
  <Badge intent="critical">critical</Badge>
</Flex>
```

## Badge Options

### Rounded

Ends of badge are fully-rounded instead of just receiving a slight radius without `rounded` property.

```jsx
<Badge rounded>Rounded Badge</Badge>
```
