title: Badge
propsOf: Badge
---

`Badge` has its default values as size small and background color as purple. The styles can be updated for size and color, and it has flags for `rounded` border and `transparent` background.

## Badge Sizes

```jsx
<Badge mr="small">small</Badge>
<Badge mr="small" size="medium" >
  medium
</Badge>
<Badge mr="small" size="large" >
  large
</Badge>
```

## Badge Intent

```jsx
<Badge intent="plain" mr="small" size="medium" >
  plain
</Badge>
<Badge intent="positive" mr="small" size="medium" >
  positive
</Badge>
<Badge intent="info" mr="small" size="medium" >
  info
</Badge>
<Badge intent="neutral" mr="small" size="medium" >
  neutral
</Badge>
<Badge intent="warning" mr="small" size="medium" >
  warning
</Badge>
<Badge intent="critical" mr="small" size="medium" >
  critical
</Badge>
```

## Badge Extra Flags

```jsx
<Badge mr="small" size="medium" transparent >
  transparent
</Badge>
<Badge mr="small" rounded size="medium" >
  rounded
</Badge>
```
