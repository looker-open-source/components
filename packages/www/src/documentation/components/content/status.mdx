The `<Status />` component can be used to indicate `critical` `inform` `neutral` `success` or `warning` of an item and empower users to be informed and take actions. This is acompleshed using `intent` property.

```jsx
<Grid columns={5}>
  <Status intent="critical" />
  <Status intent="inform" />
  <Status intent="neutral" />
  <Status intent="success" />
  <Status intent="warning" />
</Grid>
```
