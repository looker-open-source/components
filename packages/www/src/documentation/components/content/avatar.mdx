title: Avatar
propsOf: Avatar
---

`Avatar` is most often used to identify individual users or groups. Lens offers three variations on it:

- `AvatarUser` displays the profile image, if there is one available, or the user's initials.
- `AvatarIcon` displays an icon as the avatar, which can be selected with the icon prop, and has the `User` icon as default.
- `AvatarCombo` displayed AvatarUser or AvatarIcon and a badge with an icon connected to it.

## AvatarUser

Expects an object that is set to the `user` flag. This object should have: `avatar_url` to be the avatar image, or `first_name` and `last_name` as a secondary input.

```jsx
<Flex justifyContent="space-around" alignItems="center">
  <AvatarUser
    user={{
      avatar_url: 'https://www.fillmurray.com/150/150',
      first_name: 'Paul',
      last_name: 'Smith',
    }}
  />
  <AvatarUser
    user={{
      avatar_url: null,
      first_name: 'Paul',
      last_name: 'Smith',
    }}
  />
</Flex>
```

## AvatarIcon

Has a default icon value of `User`. That value can be changed by the use of the flag `icon`.

```jsx
<Flex justifyContent="space-around" alignItems="center">
  <AvatarIcon />
  <AvatarIcon icon="Code" />
</Flex>
```

### Avatar Options

Both `AvatarIcon` and `AvatarUser` can be styled for size abd color:

- flag `size`: default value `large`. That can be update to small, medium, large.
  If you choose the small size for `AvatarUser`, it will only render the first initial.
- flag `color`: default value is `#9785F2`. That can be updated using the flag color.

```jsx
;() => {
  const data = {
    avatar_url:
      'https://gravatar.lookercdn.com/avatar/e8ebbdf1a64411721503995731?s=156&d=blank',
    first_name: 'Paul',
    last_name: 'Smith',
  }
  return (
    <Flex justifyContent="space-around" alignItems="center">
      <AvatarIcon size="small" />
      <AvatarIcon icon="Group" size="medium" />
      <AvatarIcon icon="Code" />
      <AvatarUser user={data} size="small" />
      <AvatarUser user={data} size="medium" />
      <AvatarUser user={data} />
      <AvatarUser user={data} color="palette.blue500" />
      <AvatarUser user={data} color="palette.green500" />
    </Flex>
  )
}
```

## AvatarCombo

It's a combination of `AvatarUser` and `AvatarIcon`. Avatar provides three options to identify a user: a user profile image, user initials, or an SVG icon. If no data is passed as the user the Avatar Identity will be set to to `users` as on the `AvatarIcon` case.
The badge has two flags:
`badgeIcon` defines the icon to be placed as badge. The default value here is `User`,
`badgeColor` defined the color to be placed on badge. The default value here is `#9785F2`.

```jsx
;() => {
  const data = {
    avatar_url: 'https://www.fillmurray.com/150/150',
    first_name: 'Paul',
    last_name: 'Smith',
  }
  const noImageData = { ...data, avatar_url: null }

  return (
    <Flex justifyContent="space-around" alignItems="center">
      <AvatarCombo />
      <AvatarCombo user={noImageData} />
      <AvatarCombo user={data} />
      <AvatarCombo
        user={noImageData}
        color="palette.charcoal400"
        badgeColor="palette.red500"
        badgeIcon="LockClosed"
      />
    </Flex>
  )
}
```
