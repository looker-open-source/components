---
title: Avatar
propsOf: Avatar
---

`Avatar` is most often used to identify individual users or groups. Lens offers three variations on it:

- `AvatarUser` displays the profile image, if there is one available, or user's initials.
- `AvatarIcon` displays an icon as the avatar, which can be selected with the icon prop.
- `AvatarCombo` displayed AvatarUser or AvatarIcon and a badge connected to it.

## AvatarUser

```jsx
;() => {
  const goodData = {
    avatar_url:
      'https://media.istockphoto.com/vectors/noisy-glitch-pixelated-seamless-pattern-vector-id1051777344',
    first_name: 'John',
    id: 1,
    last_name: 'Smith',
  }
  const noImage = {
    avatar_url:
      'https://gravatar.lookercdn.com/avatar/e8ebbdf1a64411721503995731?s=156&d=blank',
    first_name: 'John',
    id: 2,
    last_name: 'Smith',
  }
  return (
    <>
      <AvatarUser m="medium" user={goodData} />
      <AvatarUser m="medium" user={noImage} />
    </>
  )
}
```

## AvatarIcon

```jsx
<AvatarIcon m="medium" />
<AvatarIcon m="medium" icon="Code" />
```

- `size`: small || medium || large - large being the default.
  - If you choose the small size for `AvatarUser`, it will only render the first initial.
- `color`: default value is `#4F2ABA` but that can be changed by the `color` prop.

```jsx
;() => {
  const data = {
    avatar_url:
      'https://gravatar.lookercdn.com/avatar/e8ebbdf1a64411721503995731?s=156&d=blank',
    first_name: 'John',
    id: 3,
    last_name: 'Smith',
  }
  return (
    <>
      <AvatarIcon m="medium" size="small" />
      <AvatarIcon m="medium" icon="LogoRings" color="#FF4500" size="medium" />
      <AvatarIcon m="medium" icon="Code" color="#6495ED" />
      <AvatarUser m="medium" color="#6495ED" user={data} />
      <AvatarUser m="medium" color="#FF4500" user={data} size="medium" />
      <AvatarUser m="medium" user={data} size="small" />
    </>
  )
}
```

## AvatarCombo

It's a combination of `AvatarUser` and `AvatarIcon`. Avatar provides three options to identify a user: a user profile image, user initials, or an SVG icon. If no data is passed as the user the Avatar Identity will be set to to `users` as on the `AvatarIcon` case. As for the badge on `AvatarCombo` the icon has no default value. So it has to be set when the Avatar is created.

```jsx
;() => {
  const data = {
    avatar_url:
      'https://media.istockphoto.com/vectors/noisy-glitch-pixelated-seamless-pattern-vector-id1051777344',
    first_name: 'John',
    id: 3,
    last_name: 'Smith',
  }

  const noImageData = {
    avatar_url:
      'https://gravatar.lookercdn.com/avatar/e8ebbdf1a64411721503995731?s=156&d=blank',
    first_name: 'John',
    id: 2,
    last_name: 'Smith',
  }
  return (
    <>
      <AvatarCombo m="medium" user={data} badge="LogoRings" />
      <AvatarCombo m="medium" user={noImageData} icon="Code" badge="Code" />
      <AvatarCombo m="medium" color="#6495ED" badge="Beaker" />
      <AvatarCombo m="medium" color="#FF4500" badge="Code" />
    </>
  )
}
```
