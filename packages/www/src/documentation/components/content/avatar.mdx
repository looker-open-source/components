`Avatar` is most often used to identify individual users or groups. To use the component there are three main ways:
`AvatarUser` where it displays an image if user has one saved, or the user's initials.
`AvatarIcon` where an Icon should be determined and that will be displayed. If on `Icon` is set the default pne is `users`.

Basic Uses

```jsx
;() => {
  const ImageData = {
    // eslint-disable-next-line @typescript-eslint/camelcase
    avatar_url:
      'https://gravatar.lookercdn.com/avatar/e8ebbdf1a644117215036eac62995731?s=156&d=blank',
    // eslint-disable-next-line @typescript-eslint/camelcase
    first_name: 'Luke',
    id: 61,
    // eslint-disable-next-line @typescript-eslint/camelcase
    last_name: 'Bowerman',
  }

  const noImageData = {
    // eslint-disable-next-line @typescript-eslint/camelcase

    // eslint-disable-next-line @typescript-eslint/camelcase
```

Avatar can be set as small medium or large, large being the default. If using AvatarUser with intials on size small there will be only the first initail displayed.

```jsx
<AvatarIcon  size="small" />
<AvatarIcon  size="medium" />
<AvatarIcon />
<AvatarUser user={bedData} size="small" />
```
