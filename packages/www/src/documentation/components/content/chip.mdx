`Chip`

## Chip

The `Chip` component offeres a closing and a disabled options also onClick capability.

```jsx
<Flex alignItems="center" justifyContent="space-around">
  <Chip>first chip</Chip>
  <Chip>this is a very long text that shows the word wrapping in a chip</Chip>
  <Chip disabled>disabled chip</Chip>
</Flex>
```

## Adding a onDelete

```jsx
;() => {
  const alertTrigger = () => alert('hello')
  return (
    <Flex alignItems="center" justifyContent="space-around">
      <Chip onDelete={alertTrigger}>clickable</Chip>
    </Flex>
  )
}
```
