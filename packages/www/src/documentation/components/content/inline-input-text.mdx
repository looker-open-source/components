## InlineInputText

Input styled to have underline follow the input. We have a placeholder for `title` in case nothing is provided.

```jsx
<Flex alignItems="center" justifyContent="space-around">
  <InlineInputText />
  <InlineInputText title="Type here..." />
</Flex>
```

Using the prop `simple` the input won't be as visible until hover or focus

```jsx
<InlineInputText simple title="Type here..." />
```
