## DateTime

The `DateTime` component will return a string with a language sensitive representation of the date and time. The specific moment can be specified or the default value will be assumed according to the default locale.

```jsx
<Flex justifyContent='space-around' alignItems='center'>
  <DateTime/>
</Flex>
```

## HideDate HideTime

```jsx
<Flex justifyContent='space-around' alignItems='center'>
  <DateTime hideTime/>
  <DateTime hideDate/>
</Flex>
```

## Specify locale and Date

Pass the Date desired as children and the locale as prop.

[list supported locales](https://docs.looker.com/admin-options/settings/users#locale)

```jsx
<Flex justifyContent='space-around' alignItems='center'>
  <DateTime locale='en-us'>{new Date(Date.UTC(2011, 1, 3, 5, 6, 2))}</DateTime>
  <DateTime locale='ar-EG'>{new Date(Date.UTC(2011, 1, 3, 5, 6, 2))}</DateTime>
</Flex>
```

## Styles

```jsx
<Flex justifyContent='space-around' alignItems='center'>
  <DateTime timeZone='America/New_York'/>
  <DateTime style='short'/>
  <DateTime />
  <DateTime style='long'/>
  <DateTime style='full'/>
</Flex>
```

## TimeZone

TimeZone needs to be a specific string based on the [IANA standard](https://www.iana.org/time-zone)

```jsx
<Flex justifyContent='space-around' alignItems='center'>
  <DateTime timeZone='America/New_York'/>
  <DateTime timeZone='Asia/Shanghai'/>
  <DateTime timeZone='Asia/Kolkata'/>
</Flex>
```
