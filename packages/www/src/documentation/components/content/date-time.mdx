## DateTime

The `DateTimeFormat` component will return a string with a language sensitive representation of the date and time. The specific moment can be specified as children or the default value will be assumed according to the default locale.

```jsx
<Flex justifyContent="space-around" alignItems="center">
  <DateTimeFormat />
</Flex>
```

## DateFormat

Use this component to show only the Date.

```jsx
<Flex justifyContent="space-around" alignItems="center">
  <DateFormat />
</Flex>
```

## TimeFormat

Use this component to show only the Date.

```jsx
<Flex justifyContent="space-around" alignItems="center">
  <TimeFormat />
</Flex>
```

## Specify locale and Date

Pass the Date desired as children and the locale as prop.

[list supported locales](https://docs.looker.com/admin-options/settings/users#locale)

```jsx
<Flex justifyContent="space-around" alignItems="center">
  <Text>
    <DateTimeFormat>{new Date(Date.UTC(2011, 1, 3, 5, 6, 2))}</DateTimeFormat>
  </Text>
  <Text>
    <DateTimeFormat locale="ar-EG" />
  </Text>
</Flex>
```

## TimeZone

TimeZone needs to be a specific string based on the [IANA standard](https://www.iana.org/time-zone)

When adding the timeZone the format will be long - look below for details on formating.

```jsx
<Flex justifyContent="space-around" alignItems="center">
  <Text>
    <DateTimeFormat timeZone="Asia/Kolkata" format="long" />
  </Text>
</Flex>
```

## Format

```jsx
<Table>
  <TableHead>
    <TableRow>
      <TableHeaderCell>SHORT</TableHeaderCell>
      <TableHeaderCell>MEDIUM - default</TableHeaderCell>
      <TableHeaderCell>LONG</TableHeaderCell>
      <TableHeaderCell>FULL</TableHeaderCell>
    </TableRow>
  </TableHead>
  <TableBody>
    <TableRow>
      <TableDataCell>
        <DateTimeFormat format="short" />
      </TableDataCell>
      <TableDataCell>
        <DateTimeFormat />
      </TableDataCell>
      <TableDataCell>
        <DateTimeFormat format="long" />
      </TableDataCell>
      <TableDataCell>
        <DateTimeFormat format="full" />
      </TableDataCell>
    </TableRow>
  </TableBody>
</Table>
```
