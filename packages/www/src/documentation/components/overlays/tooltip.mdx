A simple Tooltip component with out of the box styles and behavior.

```jsx
<Tooltip content="More things you should know...">
  {(tooltipProps) => <Button {...tooltipProps}>Hover for Tooltip</Button>}
</Tooltip>
```

```jsx
<Tooltip
  width="20rem"
  placement="right"
  textAlign="left"
  content={
    <>
      This is a tooltip with quite a bit of text. It's probably not ideal to
      have this much text in a Tooltip. Perhaps you should link to
      <Link href="#">another document &rarr;</Link>
    </>
  }
>
  {(tooltipProps) => (
    <ButtonOutline {...tooltipProps}>Tooltip with lots of text</ButtonOutline>
  )}
</Tooltip>
```

## Tooltip Placement

Tooltip allows you to specify where it should appear in relation to the target element. It accepts the following values:

1. `auto`
1. `auto-end`
1. `top-start`
1. `top`
1. `top-end`
1. `right-start`
1. `right`
1. `right-end`
1. `bottom-end`
1. `bottom`
1. `bottom-start`
1. `left-end`
1. `left`
1. `left-start`

```jsx
<Tooltip content="I'm on top" placement="top">
  {(tooltipProps) => (
    <Button {...tooltipProps}>
      Top
    </Button>
  )}
</Tooltip>

<Tooltip content="I'm on bottom" placement="bottom">
  {(tooltipProps) => (
    <Button {...tooltipProps}>
      Bottom
    </Button>
  )}
</Tooltip>

<Tooltip content="I'm on the left" placement="left">
  {(tooltipProps) => (
    <Button {...tooltipProps}>
      Left
    </Button>
  )}
</Tooltip>

<Tooltip content="I'm on the right" placement="right">
  {(tooltipProps) => (
    <Button {...tooltipProps}>
      Right
    </Button>
  )}
</Tooltip>
```
