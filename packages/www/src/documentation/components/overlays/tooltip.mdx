A simple Tooltip component with out of the box styles and behavior.

```jsx
;() => {
  CustomizableTooltipAttributes.zIndex = 1
  return (
    <Tooltip content="More things you should know...">
      {(eventHandlers, ref) => (
        <Button ref={ref} {...eventHandlers}>
          Hover for Tooltip
        </Button>
      )}
    </Tooltip>
  )
}
```

```jsx
<Tooltip
  content={
    'Lorem ipsum dolor amet artisan meditation four loko poutine pinterest meh cold-pressed flexitarian vaporware umami kale chips selvage salvia waistcoat occupy. Jianbing jean shorts VHS austin bushwick.'
  }
>
  {(eventHandlers, ref) => (
    <Button ref={ref} {...eventHandlers}>
      Hover me! (with lots of text)
    </Button>
  )}
</Tooltip>
```

## Tooltip Placement

Tooltip allows you to specify where it should appear in relation to the target element. It accepts the following values:

1. `auto`
1. `auto-end`
1. `top-start`
1. `top`
1. `top-end`
1. `right-start`
1. `right`
1. `right-end`
1. `bottom-end`
1. `bottom`
1. `bottom-start`
1. `left-end`
1. `left`
1. `left-start`

```jsx
<Tooltip content="I'm on top" placement="top">
  {(eventHandlers, ref) => (
    <Button ref={ref} {...eventHandlers} mr="small">
      Top
    </Button>
  )}
</Tooltip>
```
