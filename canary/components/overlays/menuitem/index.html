MenuItem

STABLE

MenuItem is an list item containing a button for triggering actions from with in a Menu or MenuList.

For accessibility reasons, if you want your MenuItem to link somewhere then you can use the itemRole prop to identify it as a link. Note: If using the props itemRole don't use the prop disabled as it is bad-form and defeats the purpose.

An icon can optionally be assigned to each item via the icon property.

Note: itemRole="link" and disabled cannot be combined (use itemRole="button" if you need to offer a disabledMenuItem`)

  • Away
<MenuList>
  <MenuItem icon={<Home />}>Home</MenuItem>
  <MenuItem
    itemRole="link"
    target="_blank"
    icon={<Public />}
    href="https://google.com"
  >
    Away
  </MenuItem>
</MenuList>

Detail & Description

Use the detail and/or description prop to provide extra explanatory information on a given item.

Tooltip

To implement a tooltip on a MenuItem, wrap the MenuItem in a Tooltip element.

<MenuList>
  <Tooltip content="It is gouda!" placement="left">
    <MenuItem>Gouda</MenuItem>
  </Tooltip>
  <MenuItem
    tooltip="I have a Swiss Cheese joke, but it's full of holes."
    tooltipPlacement="right"
  >
    Swiss
  </MenuItem>
</MenuList>
